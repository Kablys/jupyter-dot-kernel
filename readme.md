# Dot Kernel

The dot kernel for jupyter!

![](./readme_assets/demo.jpeg)

## Install

```
pip install dot_kernel
```

Run using:

```
jupyter notebook
```

Select new -> dot

![](readme_assets/new_dot.png)

and try run this:

```
 // The graph name and the semicolons are optional
 graph graphname {
     a -- b -- c;
     b -- d;
 }
```

You will see:

![](readme_assets/first_try.png)
